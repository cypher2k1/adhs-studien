Navigation im Kurs
Die Schaltflächen am linken Bildschirmrand dienen zur Navigation im Kurs oder rufen Servicefunktionen wie Hilfe, Notizblatt, Volltextsuche und Lesezeichen auf.
 
Ruft das Inhaltsverzeichnis des Kurses auf. Zunächst wird die Kapitelstruktur des Kurses angezeigt. Wenn Sie auf das Ordnersymbol zu einem Kapitel klicken, werden auch die Unterkapitel angezeigt. Mit einem weiteren Klick auf das Ordnersymbol schließen Sie die Anzeige der Unterkapitel. Mit einem Klick auf einen Kapitel-Titel oder Unterkapitel-Titel verzweigen Sie direkt in den entsprechenden Kursausschnitt. Haben Sie ein Kapitel/Unterkapitel vollständig bearbeitet, wird es entsprechend markiert. So können Sie jederzeit Ihren aktuellen Kursbearbeitungsstand ablesen.
 
Ruft das Indexverzeichnis des Kurses auf. Im Index werden alle Schlagwörter dieses Kurses aufgeführt. Jedem Eintrag sind Links zugeordnet. Diese ermöglichen Ihnen den direkten Sprung auf die Kursseiten, die inhaltlich mit dem gewählten Eintrag in Beziehung stehen.

Ruft ein Verzeichnis mit Glossareinträgen auf. Im Glossar werden wichtige Fachbegriffe des Kurses erläutert.

Ruft ein bibliografisches Verzeichnis mit empfohlener Literatur auf.

Ruft eine Lesezeichenfunktion auf. Durch Anklicken von "Neues Lesezeichen setzen" setzen Sie einen Marker auf die aktuelle Seite. Diese Seite wird damit in die Lesezeichenliste eingetragen und kann später direkt angewählt werden.
Durch Anklicken von "Löschen" entfernen Sie einzelne Einträge, mit der Schaltfläche "Alle Lesezeichen löschen" löschen Sie alle aufgelisteten Lesezeichen. Über die Schaltfläche mit dem Drucker-Symbol können Sie die Liste der eingetragenen Lesezeichen ausdrucken.


Ruft ein Notizblatt auf. Hier können Sie Ihre persönlichen Anmerkungen zum Kurs allgemein oder zu den einzelnen Kapiteln eintragen.Mit den Schaltflächen "Notiz speichern" / "Notiz löschen" speichern bzw. löschen Sie die Notizen zum Kurs oder zum angewählten Kapitel. Mit "Alle Notizen löschen" werden Ihre gesamten Notizen zum Kurs und zu allen Kapiteln gelöscht. Gespeicherte Notizen können Sie für Ihre Unterlagen ausdrucken. Über die Schaltfläche mit dem Drucker-Symbol gelangen Sie in die Druckvorschau. Durch (Doppel-)Klick auf "Notiz-Eintrag drucken" erteilen Sie den Druckauftrag.
Wichtiger Hinweis: Die Funktion Notizblatt ist nur für kürzere Notizen gedacht. Lange Texte können hier nicht gespeichert werden. In diesem Fall erhalten Sie eine entsprechende Rückmeldung.
Möchten Sie Ihre Notizen durchsuchen, können Sie Ihre(n) Suchbegriff(e) in das Textfeld "Suchen nach" eingeben. Durch (Doppel-)Klick auf den "Lupe"-Button oder durch Drücken der Return- bzw. Enter-Taste wird die Suche gestartet. Das Suchergebnis wird in einem Extrafenster angezeigt und kann direkt angewählt werden.

Ruft eine Suchefunktion für alle Texte des Kurses auf. Hier können Sie Ihre(n) Suchbegriff(e) in das Textfeld "Im Kurs suchen nach" eingeben und entweder durch (Doppel-)Klick auf den "Lupe"-Button oder durch Drücken der Return- bzw. Enter-Taste den gesamten Kurs im Volltext durchsuchen. Die Suchergebnisse (max. 40 Treffer) können direkt angewählt werden. Durch Überschreiben des Suchbegriffs und (Doppel-)Klick auf den "Lupe"-Button oder durch Drücken der Return- bzw. Enter-Taste starten Sie die Volltextsuche erneut.

Ruft diese Hilfetexte zur Bedienung des Kurses auf.

Gibt Informationen zu den Produzenten dieses Kurses.

Ruft die sachlogisch vorausgehende Seite eines Kapitels auf. Um auf Ihren persönlichen Kurspfad zurückzugehen, nutzen Sie bitte die Zurück-Funktion des Browsers.

Ruft die sachlogisch folgende Kursseite auf.


Navigation auf einer Kursseite
Auf der Kursseite wird der Lehrtext mit Quellcode-Grafiken, Abbildungen und vertonten Animationen dargestellt. Über Schaltflächen haben Sie die Möglichkeit, ergänzende Informationen, längere Programmcodes, Applets und unterstützende Screenshots aufzurufen. Die Schaltflächen sind entsprechend gekennzeichnet.

Außerdem finden Sie auf der Kursseite die folgenden Formatierungen:
Fettdruck für Hervorhebungen
Kursivstellung für Eigennamen und englische Begriffe
Kapitel 1.4 für interne Verweise, die auf das angegebene Sprungziel verzweigen
www.url.de für externe Verweise, die die angegebene Internetadresse (online) aufrufen, und für Verweise auf die Dokumentation (CD-ROM).
Java für Glossareinträge, die das Glossar dieses Kurses aufrufen
Courier New für Programmier-Syntax, Dateinamen un Verzeichnisse

Zur besseren Lesbarkeit ist die Java-Syntax wie folgt formatiert:
public für Schlüsselworte
"Hallo" für Zeichenketten, die ausgegeben werden
println für die allgemeine Java-Syntax sowie für andere, im Kurs verwendete Programmiersprachen wie XML, SQL etc.

Pfadangaben für Beispielprogramme und Praxisbeispiele
 

Bsp03Strings.java 		   
Bsp03Strings.java
  Im Kurs werden in der Regel nur die charakteristischen Abschnitte oder Zeilen erklärt. Die vollständigen Beispielprogramme sind auf der CD-ROM im Ordner work abgelegt. Für jedes Kapitel gibt es einen Unterordner (z.B. Kap1 steht für Kapitel 1, Kap2 steht für Kapitel 2, usw.), in dem sich die angegebenen Dateien (hier im Beispiel ist es die Datei Bsp03Strings.java) befinden. Im Kurs können Sie durch Klick auf die Pfadangabe eine Text-Version (txt) der entsprechenden Datei einsehen. Programmierbeispiele zum Üben sind als Praxisbeispiele ausgewiesen.

Hinweis:
Wenn Sie die abgelegten Beispielprogramme bearbeiten möchten, funktioniert dies auf der CD-ROM wegen des Schreibschutzes verständlicherweise nicht. Sie sollten daher die Dateien auf Ihre Festplatte kopieren. Für die Bearbeitung der Beispiele ist es egal, in welches Verzeichnis Sie die Dateien ablegen. Im Kurstext wird als Platzhalter meist das Verzeichnis d:\work verwendet. Dabei gibt es ein kleines Problem: Dateien, die von der CD-ROM auf die Festplatte kopiert werden, erhalten automatisch einen Schreibschutz. Damit können die Quelltexte nicht mehr verändert und kompiliert werden. Aus diesem Grund finden Sie im Ordner work eine ausführbare Datei (work.exe), die Ihnen den kompletten work-Ordner auf Ihre Festplatte kopiert.

Jeweils am Ende eines Kapitels befinden sich Übungen zur Wissensüberprüfung. Über Schaltflächen können Sie Ihre Eingabe auswerten, die Lösung abrufen oder Ihre Eingabe löschen. Unter jeder Übung finden Sie Links, die auf inhaltlich relevante Kursseiten verzweigen.
