Was ist Java?

Programmiersprachen gibt es viele: ganz spezielle Sprachen für ganz spezielle Anwendungen und allgemeine "Mainstream"-Sprachen.

    

Java, ursprünglich entwickelt von der Firma SUN (seit 2010 hat Oracle die Weiterentwicklung übernommen), eignet sich hervorragend für das Schreiben von Programmen in allen aktuellen "Problemgebieten". So sollen Programme häufig unabhängig von unterschiedlichen Plattformen (Betriebssystemen, Mikroprozessoren) sein und sich über mehrere Rechner in einem Netzwerk verteilen lassen.

  
Definition

Beide Aspekte werden von Java als Programmiersprache und der damit verbundenen Architektur (virtuelle Maschine) glänzend unterstützt.
