Auf welchen Betriebssystemen und Rechnern läuft Java?

Im Wesentlichen gibt es Java-Entwicklungsumgebungen für folgende Systeme:

    
  Windows
  MacOS
  LINUX
  Diverse Unix-Implementierungen wie z.B. Solaris
