Die Entwicklungsumgebung für Java

Zum Programmieren in Java brauchen Sie neben einem Texteditor zum Eintippen des Programms noch einen Übersetzer (engl.: Compiler), der Ihren Text in eine ausführbare Form bringt. Ein solcher Compiler ist Bestandteil des Java Development Kits (JDK) der Firma Oracle.

    

Ein erster Schritt des Programmierens in Java ist also die Installation des JDK. Diese beginnt mit dem Download aus dem Internet - Sie finden auch eine aktuelle Version des JDK auf der CD zu diesem Kurs.

    
Startpunkt für den Download des JDK ist üblicherweise die Java WebSite der Firma Oracle. Die Internetadresse lautet:

http://java.com

    

Zunächst werden wir Ihnen nun zeigen, wie Sie eine Entwicklungsumgebung für Java (das Java Development Kit von Oracle) auf Ihrem Rechner installieren können.
