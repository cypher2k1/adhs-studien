Entwicklungsumgebung - Eclipse
Eclipse ist eine frei verfügbare Open-Source-Entwicklungsumgebung, deren Entwicklung und Verbreitung von IBM vorangetrieben wird. Die Quelle lautet:    

http://www.eclipse.org/eclipse/

    

Die Entwicklungsumgebung, die selbst in Java programmiert ist, besteht im Wesentlichen aus einem Editor und einer Reihe von Plug-Ins für die Programmierung in Java, C++, PHP etc. Verschiedene Assistenten erleichtern das Programmieren in Java und das Anlegen neuer Projekte, neuer Dateien oder Klassen innerhalb von Projekten.
