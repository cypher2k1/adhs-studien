Falls Sie sich beim Eingeben des Programms vertippt haben, erscheint jetzt eine Fehlermeldung. In diesem Fall sollten Sie Ihren Quelltext noch einmal mit der Vorgabe vergleichen, den oder die Fehler korrigieren, und es erneut versuchen.

    

Auch bei einer fehlerfreien Eingabe ist das Ergebnis am Anfang vielleicht etwas enttäuschend, denn zunächst einmal passiert nichts, zumindest nichts Sichtbares. Allerdings wird vom Java-Compiler in dem aktuellen Verzeichnis d:\work die Datei GanzEinfach.class erzeugt, die den Java-Bytecode enthält. Schauen Sie doch mal nach! Diesen Code können Sie im nächsten Schritt ausführen.
