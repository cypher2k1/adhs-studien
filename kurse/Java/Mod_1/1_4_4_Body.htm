Ganz einfach! (3)

Um den Java-Bytecode auszuführen, benutzen Sie das JDK-Tool java, die virtuelle Maschine von Java.

  
Praxisaufgabe

Als Parameter übergeben Sie dem Programm einfach den Namen Ihrer class-Datei, die den Java-Bytecode enthält.

   

Et violà, es erscheint der gewünschte Text in dem Fenster.

    
  So sieht es aus.
   


Der erzeugte Bytecode ist übrigens nicht an Ihren Computer oder Ihr System gebunden. Sie können ihn auf jeder Plattform verwenden. Das heißt, Sie können die Datei GanzEinfach.class auch auf einen Apple Macintosh oder ein LINUX-System kopieren und Ihr Programm dort ausführen - sofern dort die nötige Unterstützung für Java installiert ist.

    

Noch eindrucksvoller wird dies, sobald Ihre Programme nicht nur einen einfachen Text ausgeben, sondern mit Unterstützung des AWT oder der Swing-Klassen eine grafische Benutzeroberfläche besitzen.

    

Beide Systeme werden im Verlaufe dieses Kurses noch genauer erläutert.
