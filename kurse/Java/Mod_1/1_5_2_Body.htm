Guten Tag! Wie heißen Sie?

Das folgende Programm ist ein wenig komplizierter als das letzte: Es gibt nicht nur einen Text auf dem Bildschirm aus, sondern lässt den Benutzer auch Text eingeben, der auf dem Monitor erscheint. Je nach Benutzereingabe wird am Ende erneut ein Text ausgegeben.

    

Klicken Sie auf den neben stehenden Link und geben Sie diesen Quellcode in einem Texteditor Ihrer Wahl ein. Speichern Sie es in einer entsprechenden Datei. Bei einem Blick auf das Programm sollten Sie jetzt wissen, wie die Datei heißen muss.

  
GutenTag.java

  So sieht es aus.
   

Nachdem Sie den Quelltext in der Datei GutenTag.java abgespeichert haben (GutenTag heißt die Klasse, die in dem Quelltext definiert wird), können Sie das Programm wie gewohnt mit dem javac-Tool kompilieren, also den dazugehörenden Java-Bytecode erzeugen.

    
  So sieht es aus.
   

Falls Sie sich vertippt haben, korrigieren Sie bitte die Fehler, und probieren Sie es erneut. Wenn Sie das Programm nicht auf Anhieb verstehen: macht gar nichts!, denn schließlich beginnen wir ja erst mit dem Kurs.
