Höhere Programmiersprachen

Das Programmieren in Assembler ist recht umständlich, da die Befehle sehr kryptisch sind. Man braucht alleine schon vier oder fünf dieser Anweisungen, um zwei Zahlen zu addieren.

    

Außerdem sind die Anweisungen in Assembler oder im Maschinencode auf jedem Prozessor anders, d.h. ein Programm, das auf einem Rechner geschrieben wurde, läuft vermutlich nicht auf einem anderen Rechnertyp mit anderem Prozessor.

    

Im Laufe der Zeit wurden deshalb eine ganze Reihe von verschiedenen - so genannten "höheren" - Programmiersprachen entwickelt, die anschaulicher und leichter zu verstehen sind.

    

Das Programmieren in diesen Sprachen ist im Vergleich zu Assembler sehr viel einfacher. Und es gibt Programme, die die "höheren" Programmiersprachen nach Assembler oder in den Maschinencode übersetzen.

    

Diese Programme nennt man Compiler.

  
Definition

Und eine dieser "höheren" Sprachen ist Java - doch Java ist viel mehr als nur eine neue Programmiersprache. Mehr Informationen dazu folgen im nächsten Kapitel.
