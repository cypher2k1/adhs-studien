Objektorientiertes Programmieren - Instanzen
Wichtig ist, dass wir bisher im Grunde nur von Klassen gesprochen haben: Klassen sind zwar Typen, jedoch keine Werte. Ihr spezieller Hund, der gerade neben Ihnen sitzt und einen speziellen Namen hat, ist aber ein Wert bzw. ein Objekt bzw. eine Instanz der Klasse Hund.
 
Hinweis

Das Erzeugen eines solchen Objekts bzw. das Instanziieren der Klasse Hund geschieht in Java durch den new-Operator. Dieser wird durch das Schlüsselwort new eingeleitet, gefolgt von dem Namen der Klasse und einer (eventuell leeren) Folge von Parametern in runden Klammern.

    
 
Haustier Hund;
Hund = new Haustier();
  		 
 
Beispiel

Ergebnis dieses Operators ist immer ein Objekt der Klasse, die hinter dem Schlüsselwort new steht, zum Beispiel Hund.

    

Dieses Objekt können Sie nun einer Variablen zuweisen. Der Datentyp dieser Variablen ist die Klasse, die Sie mit dem new-Operator instanziiert haben. Dabei ist folgendes zu beachten:

    
Zum einen handelt es sich bei Klassen um Referenzdatentypen, d.h. genau genommen erhalten Sie als Ergebnis des new-Operators nicht das volle Objekt zurück, sondern nur eine Referenz darauf. Das Objekt selber steht irgendwo im Arbeitsspeicher. Allerdings ist Java relativ geschickt darin, diesen Umstand vor Ihnen zu verbergen. Dieses "technische Detail" sollten Sie aber nicht außer Acht lassen.
