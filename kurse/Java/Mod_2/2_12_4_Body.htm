Objektorientiertes Programmieren - Punktnotation

Auf die Attribute der Objekte bzw. deren Werte greifen Sie in Java mit der so genannten Punktnotation zu: Sie schreiben hinter das Objekt (zum Beispiel den Namen der Variablen) einen Punkt und dahinter den Namen des Attributs.


  
Definition
 
mein_hund.name = "Bello";
  		 
 
Beispiel

Dabei ist es egal, ob das Attribut in der Klasse des Objekts explizit definiert ist oder ob es von einer Oberklasse geerbt wird.

    
     
Objektorientiertes Programmieren - Methoden    

Ein weiterer Bestandteil der Klassen neben den Attributen sind die Methoden, die das Verhalten der Objekte beschreiben. Methoden haben im Wesentlichen einen Namen und eine Reihe von Anweisungen, die ausgeführt werden, sobald die Methode aufgerufen wird. Das Verhalten der Methode kann durch eine Reihe von Parametern gesteuert werden und als Ergebnis kann ein Wert eines bestimmten Typs an den Aufrufer der Methode zurückgeliefert werden.
