Iteratoren

Java bietet auﬂerdem so genannte Iteratoren zum Durchlaufen der Liste an. Ein Iterator ist ein Objekt, welches auf eine Liste verweist und (in gewisser Weise) mit einem Zeiger in dieser Liste ein aktuelles Element der Liste markiert. Mit diesem Zeiger lässt sich dann das "aktuelle" Element der Liste auslesen. Alle folgenden Elemente der Liste lassen sich ebenfalls auslesen, indem man zum Beispiel diesen Zeiger in der Liste immer weiter nach hinten schiebt, bis kein weiteres Element mehr vorhanden ist.

  
Definition

Konkret wird ein Iterator zu der Liste aller Klassenkameraden wie folgt erzeugt:

    
 
Iterator iter = klassenkameraden.iterator();

  
 
Beispiel

Der Iterator wird vor (!) dem ersten Element der Liste platziert. Die wichtigsten Methoden des Iterators sind:

   
  next(), die das jeweils nächste Element der Liste als aktuell markiert und dieses Element als Ergebnis zurückliefert.
  hasNext() prüft, ob es überhaupt ein nächstes Element gibt oder ob der Zeiger bereits am Ende der Liste angekommen ist.
   

Ein üblicher Durchlauf durch eine Liste sieht also wie folgt aus:

    
  Zunächst wird ein Iterator zu der Liste erzeugt.
  Anschließend wird eine while-Schleife konstruiert, die solange durchlaufen wird, wie es nächste Elemente in der Liste gibt.
  Innerhalb der Schleife wird dann das jeweils nächste Element mit der next()-Methode aus der Liste ausgelesen. Diese Methode liefert ein Objekt der Klasse java.lang.Object zurück, das man gegebenenfalls in den korrekten Datentyp der Listenelemente castet, beispielsweise in Strings.

   

Die Liste aller Klassenkameraden lässt sich nun zum Beispiel wie folgt auf dem Bildschirm ausgeben:

    
 
Iterator iter = klassenkameraden.iterator();
while( iter.hasNext() ){
      String name = (String) iter.next();

      System.out.println( name );
}
