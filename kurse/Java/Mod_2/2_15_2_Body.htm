Übersicht über Wrapper-Klassen

In Kapitel 2.1 haben Sie bereits elementare Datentypen kennen gelernt, zum Beispiel boolean, int oder double: So kann eine Variable vom Type int genau einen Wert haben, zum Beispiel 0, 5, 17 oder 35. Oder einer Variable vom Typ boolean hat immer entweder den Wert true oder den Wert false.

    


Abbildung: Elementarer Datentyp

    
Zu diesen Datentypen bietet Java so genannte Wrapper-Klassen: Objekte solcher Klassen enthalten je einen Wert eines elementaren Datentyps, bieten aber zusätzliche Vorteile:  
Definition
  Neben dem eigentlichen Wert wie 0, 5, 17 oder 35 bieten Wrapper-Klassen zusätzliche Methoden an, um beispielsweise einen String in eine Zahl zu konvertieren.
  Elementare Datentypen können keinen "noch nicht vorhandenen" Wert abbilden. So sind Werte vom Typ int immer 0, 1, 2, 3 etc., aber es gibt keinen Wert, mit dem man Lottozahlen angeben kann, die noch nicht gezogen, oder Temperaturen, die noch nicht gemessen wurden. Bei Wrapper-Klassen gibt es hingegen den Wert null, mit dem ein nicht vorhandener Wert beschrieben werden kann.
  Im Unterschied zu elementaren Datentypen können sie in Listen-Klassen (vgl. Kapitel 2.13) verwendet werden.

   
Die Wrapper-Klassen sind in dem Package java.lang definiert.
