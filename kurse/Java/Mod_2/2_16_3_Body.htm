public, private und protected

In Java wird die Sichtbarkeit nach außen von Klassen, Attributen und Methoden durch Angabe der Schlüsselwörter public, private und protected gesteuert.

  
Definitionen

Die bisher benutzen Klassen waren alle public, d.h. sie waren immer überall sichtbar. Klassenelemente, die als private definiert werden, sind nur in der aktuellen Datei sichtbar bzw. im aktuellen package.

    

Auch alle bisher verwendeten Attribute waren public, denn auf sie konnte aus allen anderen Klassen und Methoden zugegriffen werden. Diese Sichtbarkeit kann mit dem Schlüsselwort private für andere Klassen eingeschränkt werden. Mit Ausnahme der eigenen Klasse bleibt dieses Attribut anderen Klassen verborgen.

    

Mit dem Schlüsselwort protected schließlich kann der Zugriff auf ein Attribut auf diejenige Klasse, in der das Attribut verwendet wird und auf alle davon abgeleiteten Klassen eingeschränkt werden. Von außen kann keine Klasse, die nicht von dieser Klasse abgeleitet ist, auf das Attribut zugreifen.
