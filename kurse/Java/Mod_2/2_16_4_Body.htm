Konstruktoren

Zu jeder Klasse können Konstruktoren definiert werden. Es sind spezielle Methoden, die genauso heißen wie die Klasse, innerhalb derer sie definiert werden. Sie dürfen keinen Wert als Ergebnis zurückgeben.

  
Definition

Sie stellen lediglich Parameter zu Verfügung, die eine neue Instanz der Klasse initialisieren: Konstruktoren werden nicht explizit aufgerufen, sondern implizit beim Aufruf des new-Operators.

    

So können Sie beim Erzeugen einer Instanz der Klasse Rechteck einen einzelnen Parameter für die Kantenlänge angeben, um ein Quadrat zu erzeugen, oder zwei Parameter für die Erzeugung eines normalen Rechtecks.

    
Voraussetzung dafür ist, dass derjenige, der die Klasse erstellt hat, genau diese Konstruktoren auch vorgesehen und definiert hat.
