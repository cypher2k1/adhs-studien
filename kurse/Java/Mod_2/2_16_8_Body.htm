Mehrfachvererbung in Java - Interfaces (2)

Wenn eine Klasse ein Interface implementiert, müssen für alle Methoden des Interfaces Implementierungen angegeben werden. Falls nicht alle Methoden implementiert werden, ist die Klasse abstrakt, d.h. es können keine Objekte dieser Klasse erzeugt werden. Möglich ist aber, Unterklassen zu definieren, in denen die Methoden implementiert werden.

    

So kann zum Beispiel die Klasse quadrat als Implementierung des Interfaces form formuliert werden:


    
 
class quadrat implements form {

   private int kantenlaenge;

   public int umfang() { return 4 * kantenlaenge; }
   public int inhalt() { return kantenlaenge * kantenlaenge; }
}
