Garbage Collection

In Java werden Instanzen von Klassen durch den new-Operator erzeugt:

    
 
Rechteck r = new Rechteck(5, 10); 		 
 
Quadrat q = new Quadrat(7);

 
 
Beispiel


Dadurch wird im Arbeitsspeicher Platz reserviert und mit den entsprechenden Attributwerten gefüllt. Allerdings gibt es keine Möglichkeit, diesen Speicher explizit wieder freizugegeben, um ihn anderweitig zu verwenden.

    

Java benutzt die Garbage Collection, um nicht mehr benötigten Speicherplatz freizugeben. Und nur Java entscheidet, wann ein Speicherbereich nicht mehr benötigt wird.
