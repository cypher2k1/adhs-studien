Exception-Handling - Definition

Nobody is perfect! Und wie im richtigen Leben auch, kann beim Ausführen von Programmen etwas schief gehen. Zum Beispiel versucht Ihr Programm, den Inhalt einer Datei zu lesen, die nicht existiert, in eine Datei zu schreiben, obwohl die Festplatte voll ist, oder eine unerlaubte Division mit den Werten zweier Variablen durchzuführen.

    

Als Prophylaxe können in Java mit dem Exception-Handling Anweisungen spezifiziert werden. Diese werden in Notfällen ausgeführt, um als eine Art Rettungsfallschirm für eine sanfte Landung zu sorgen, den Aufprall beim Absturz zu mindern und an geeigneter Stelle eventuell die Ausführung des Programms fortzusetzen.

  
Definition

Alle Ausnahmesituationen werden durch einzelne Objekte (Instanzen) abgebildet. Jede dieser Exceptions ist Instanz einer Klasse, die von Exception abgeleitet ist. Für verschiedene dieser Ausnahmesituationen gibt es in Java unterschiedliche Klassen.
