Datentypen - Elementare Datentypen

Java bietet die folgenden elementaren Datentypen:

    
  Typ  bytes    Wertebereich
 boolean   1   false, true 
 byte   1   -128 bis 127 
 short   2   -32.768 bis 32.767 
 int   4   ca. -2 Mrd. bis ca. 2 Mrd. 
 long   8   -263 bis 263-1 
 char   2   0 bis 65.535 als UniCode Zeichen 
 float   4   ca. +/- 3*1038  
 double   8  ca. +/- 1*10308 
 
Definition

Referenzdatentypen von Java sind alle Klassen und Interfaces, sowie Felder (Arrays) und Strings.

    

Mehr zum Thema Klassen finden Sie in Kapitel 2.10.
