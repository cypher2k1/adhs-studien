String - Definition

Zeichenketten (Strings) sind die ersten komplexen Datentypen. Sie werden in Java als Referenzdatentypen behandelt. Der eigentliche Wert eines Strings ist also nicht die Kette der enthaltenen Zeichen, sondern ein Verweis auf diese Kette.

  
Definition

Der Bereich des Arbeitsspeichers, der von der Zeichenkette belegt wird, wird alleine durch Java verwaltet. Nur Java ist dafür verantwortlich, nicht mehr benötigten Speicherplatz zu löschen.

    

Wie der String aussieht, der ja eigentlich ein Verweis ist, bleibt vor dem Programmierer verborgen. Diese Verweise entsprechen damit nicht ganz den Pointern, die Sie vielleicht von C oder C++ kennen. Dementsprechend gibt es in Java auch keine Pointerarithmetik. Sie können einen Pointer nicht verschieben.

    

Strings sind in Java Instanz der gleichnamigen Klasse und müssen zunächst, wie alle anderen Instanzen auch, durch den new-Operator erzeugt werden. Dafür gibt es eine Art Short-Cut. Schreibt man einen String in Anführungszeichen, wird bei der Zuweisung ein entsprechendes Objekt erzeugt.

    
In Java können Sie mit Strings genauso arbeiten wie mit anderen Datentypen auch. Sie können Strings einer Variablen zuweisen oder Strings durch Addition zusammenfügen, d.h. hintereinander hängen.
 
Hinweis

Für Strings werden außerdem eine Reihe von Methoden definiert, zum Beispiel, um die Stringlänge oder die Anzahl der Zeichen zu berechnen.
