StringBuffer

Strings lassen sich in Java im Grunde nicht manipulieren. Immer wird ein neuer String erzeugt, statt den alten zu verändern. Der neue String setzt sich beispielsweise aus dem alten String plus etwas Neuem zusammen. Oder er entsteht durch das Weglassen eines Teilstrings. Der alte String wird durch Java automatisch "entsorgt". Für die meisten Anwendungen reicht dies vollkommen aus.
Das automatische Löschen der Strings übernimmt die Garbage Collection. Mehr Infos dazu gibt es in Kapitel 2.14.

    

Für alle anderen Fälle gibt es in Java die StringBuffer. Diese können während eines Programms erweitert oder gekürzt werden. Es stehen Methoden zur Verfügung, mit denen die einzelnen Zeichen des StringBuffers modifiziert werden können. Jeder StringBuffer muss aber in einen String konvertiert werden, damit er auch so verwendet werden kann.
