Array

Ein Array wird in Java durch eckige Klammern beschrieben. Dabei müssen alle Werte des Arrays den gleichen Datentyp haben. So ist int ein einfacher Datentyp für Zahlen und
int [] ein Array von Zahlen, zum Beispiel für die Lottozahlen vom letzten Samstag.

  
Definition

Man spricht in diesem Fall von einem eindimensionalen Array. Mehrdimensionale Arrays sind in Java auch möglich: int [] [] ist ein Datentyp für ein Array von einem Array von Zahlen, zum Beispiel den Lottozahlen des letzen Jahres.

    
Da es sich bei Arrays auch um Referenzdatentypen handelt, müssen sie zunächst mit dem new-Operator erzeugt werden.
 
Hinweis

Mehr über den new-Operator finden Sie in Kapitel 2.12.

    
Auf die einzelnen Elemente eines Arrays kann zugegriffen werden, indem die Nummer des gewünschten Elements in eckige Klammern gesetzt wird.

    

Für den Zugriff auf die Elemente der zwei- oder mehrdimensionalen Arrays müssen die eckigen Klammern zweimal hintereinander geschrieben werden.
