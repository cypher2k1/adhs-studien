if-Anweisung - Definition
Die if-Anweisung dient dazu, bestimmte Anweisungen nur dann auszuführen, wenn eine bestimmte Bedingung erfüllt ist. Die Anweisung setzt sich zusammen aus:  
Definition
   Schlüsselwort if
   Wert oder Ausdruck
   Anweisungen
 

Der Datentyp des Werts bzw. des Ausdrucks muss boolean sein. Der Wert oder Ausdruck steht in runden Klammern. Die Folge von Anweisungen wird in geschweifte Klammern eingeschlossen. Gibt es nur eine einzelne Anweisung, kann auf die geschweiften Klammern verzichtet werden.

    

Bei Bedarf kann außerdem eine Folge von Anweisungen angegeben werden, die ausgeführt werden sollen, falls die angegebene Bedingung nicht erfüllt ist. Java nennt dies den else-Zweig der if-Anweisung.

    

  if (Ausdruck oder Wert) {  

     Anweisung 1;    

  }

  else  
     Anweisung 2;
