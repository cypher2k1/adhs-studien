switch-Anweisung
Oft treten viele if...else-Anweisungen hintereinander auf, was schnell unübersichtlich wird. In manchen dieser Fälle kann die switch-Anweisung das verhindern.
 
Hinweis

Die switch-Anweisung vergleicht einen bestimmten Ausdruck oder Wert vom Typ char, byte, short, int oder long mit anderen Werten oder Ausdrücken. Für den Fall, dass der Wert mit einem der Vergleichswerte übereinstimmt, werden Anweisungen angegeben, von denen die entsprechenden schließlich ausgeführt werden.

  
Definition
Die switch-Anweisung von Java besteht aus:    
  Schlüsselwort switch
  Ausdruck
  case-Anweisungen mit Konstanten (Vergleichswerte für die switch-Anweisung)
  Anweisungen
   


  switch (Ausdruck) {  

     case constant:    
        Anweisung;    
     ...
  }
