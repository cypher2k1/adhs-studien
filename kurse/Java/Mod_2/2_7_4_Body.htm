switch-/case-Anweisung


Abbildung: switch-/case-Anweisung
  
In Abhängigkeit davon, welchen Wert a genau hat, wird mindestens eine der Anweisungen x1, x2 ... etc. ausgeführt.

   
Bitte beachten Sie, dass nach x1 auch alle weiteren Anweisungen ausgeführt werden, wenn am Ende von x1 keine break-Anweisung steht.
