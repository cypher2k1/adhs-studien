do-Anweisung

Ähnlich wie die while-Schleife führt die do-Anweisung eine Folge von Anweisungen solange aus, bis eine bestimmte Bedingung erfüllt ist. Im Gegensatz dazu wird jedoch die Bedingung erst geprüft, nachdem die Anweisungen ausgeführt wurden.

  
Definition
Die Anweisungen werden also mindestens einmal ausgeführt. Erst dann wird geprüft, ob die Bedingung erfüllt ist. Wenn ja, werden die Anweisungen abgearbeitet und die Bedingung erneut überprüft.
 
Hinweis
Die do-Anweisung von Java besteht aus:    
  Schlüsselwort do
  Anweisung(en)
  Bedingung am Ende der Anweisung(en)
   



  do {

      Anweisung;

  } while (Bedingung);
