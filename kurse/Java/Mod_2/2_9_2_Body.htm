break-Anweisung

Die break-Anweisung beendet innerhalb einer while-, do- oder for-Anweisung das Ausführen eben dieser Anweisung, um mit den Anweisungen hinter der while-, do- oder for -Anweisung fortzufahren.

  
Definition

So kann eine while-Schleife, die bis 10 durchlaufen sollte, auch bei 5 abgebrochen werden. Die Anweisungen innerhalb der Schleife werden dann nicht mehr ausgeführt.

    

Hier ein Beispiel für die oben beschriebene break-Anweisung:

    
 
int i;

i=1;
while (i <= 10) {

  System.out.println (i);  

  if (i == 5)
       break;

  i ++;
}
