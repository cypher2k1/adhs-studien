Wie kommen Applets auf eine Seite?

Applets werden mit dem <applet>-Tag in die HTML-Seiten eingebunden. Analog zur Implementierung eines Bildes steht das Applet selber in einer eigenen Datei (mit der Endung .class). Die Adresse und der Dateiname stehen in dem Attribut code. Applet-Höhe und -breite werden über height und width definiert.

    

Im Gegensatz zum <img>-Tag gibt es für das Applet ein End-Tag: </applet>. Zwischen einleitendem und abschließendem Tag stehen Parameter, die das Verhalten des Applets auf der Internetseite steuern.

    

Mit <param> werden Parameter aus der HTML-Seite an ein Applet übergeben. Dazu erhält jedes Tag zwei Attribute: name und value. Der angegebene Wert wird als String unter dem entsprechenden Namen an das Applet übergeben:

    
 
<param name="farbe" value="#8888ff">

 
 
Beispiel


Der String #8888ff unter dem Namen farbe wird als Parameter an das Applet übergeben und kann dort mit der Methode getParameter() ausgelesen werden:

    
 
farbe = getParameter ("farbe");
