Kein Java - was nun?
Nicht alle Browser unterstützen Java - und bei manchen Browsern lässt sich die eingebaute Unterstützung für Java auch abschalten.
 
Hinweis

Suchmaschinen, wie z.B. google.de, unterstützen in der Regel auch kein Java. Wenn Sie also die Navigation durch Ihre WebSite alleine Java-Applets überlassen, können solche Suchmaschinen Ihre Webseite nicht indizieren.

    

Sie sollten immer eine alternative Navigation für nicht Java-fähige Browser anbieten. Hierbei hilft folgender Umstand: Java-fähige Browser sind so programmiert, dass sie alles mit Ausnahme der <param>-Angaben ignorieren, was zwischen <applet> und </applet> steht.

    

Das heißt, wenn Sie dort einen dezenten Hinweis auf das Fehlen von Java unterbringen, oder eine Alternative für die Navigation durch die Seiten, wird ein Java-fähiger Browser dies nicht berücksichtigen.

    

Ansonsten gilt, dass Browser alles ignorieren, was sie nicht kennen. Ein nicht Java-fähiger Browser wird die <applet>, </applet> und <param>-Tags ignorieren, aber alles anzeigen, was eventuell dazwischen steht: also genau den Hinweis oder die Navigation.
