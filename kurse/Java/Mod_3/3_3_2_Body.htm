Applet und Malfunktionen

Die paint()-Methode eines Applets wird vom Browser aufgerufen, wenn die Fläche des Applets neu gezeichnet werden soll - weil die Applet-Seite gerade erst aufgerufen wurde oder durch ein anderes Fenster verdeckt war.

    

Als Parameter liefert der Browser dem Applet eine Umgebung vom Typ Graphics. Diese Umgebung bietet eine ganze Reihe von Grafikoperationen zum Malen an. Beispiele sind:

    
   drawArc()
   drawImage()
   drawLine()
   drawOval()
   drawPolygon()
   drawPolyline()
   drawRect()
   drawRoundRect()
   drawString()
   fillArc()
   fillOval()
   fillRect()
   


Außerdem können Sie mit setColor() die Farbe des Pinsels bestimmen und mit setFont() einen Font auswählen.
