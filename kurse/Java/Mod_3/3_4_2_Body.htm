Threads

Eine der Stärken von Java liegt darin, dass bestimmte Bereiche, die in anderen Programmiersprachen und Systemen sehr umständlich zu programmieren sind, sehr elegant unterstützt werden. Gemeint sind die API (Application Programming Interface). Hierzu gehören die Threads - Programmteile, die "zur selben Zeit" ausgeführt werden.
