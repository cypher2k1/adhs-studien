Die run()-Methode (3)

Wenn ein neues Bild einer Animation berechnet wurde, kann die repaint()-Methode des Applets aufgerufen werden. Sie veranlasst den Browser, den Bereich des Applets neu zeichnen zu lassen.

    

Der Browser ruft dann automatisch die paint()-Methode des Applets auf - und nicht das Applet selber. Genau genommen ruft der Browser die update()-Methode des Applets auf. Sie löscht den Hintergrund und aktiviert zum Neuzeichnen die paint()-Methode.


    
 
public void run() {

   while(true) {

      try { thread.sleep(1000); }
      catch(InterruptedException e) {}

      [ ... ]
      repaint();
   }
}
