Die run()-Methode (4)

Das Applet der folgenden Animation kennen Sie bereits aus Kapitel 1.11. Hier nun wechselt die Hintergrundfarbe automatisch einmal pro Sekunde von Blau auf Schwarz und zurück.


    
 
public void run() {

   while(true) {

      try { thread.sleep(1000); }
      catch(InterruptedException e) {}

      if((farbe.compareTo( "#000000") == 0))
         farbe = "#8888ff";
      else
         farbe = "#000000";

      repaint();
   }
}
