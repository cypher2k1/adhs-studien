Applets - Bilder laden

Bilder werden in Java durch die Klasse Image in der API beschrieben. In einem Applet können Sie Bilder einladen, indem Sie die getImage() -Methode aufrufen.

    

Das fertige Applet finden Sie in Kapitel 3.4 unter Applet anzeigen.

    
 
Image fisch1;

fisch1 = getImage(getDocumentBase(), "fisch1.gif");

 
 
Animation2.java


Die zwei Parameter beschreiben die Adresse und den Namen der Bilddatei. Befindet sich die Datei in demselben Verzeichnis wie die HTML-Datei, in die das Applet eingebunden ist, können Sie die getDocumentBase()-Methode verwenden.

    

Ergebnis der getImage()-Methode ist Objekt von Typ Image.

    
Mehr Informationen zu der Klasse Applet und den Methoden getImage() und getDokumentBase() finden Sie in der JDK-Dokumentation auf der CD-ROM unter:
