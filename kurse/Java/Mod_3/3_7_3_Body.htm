Label und TextField

Labels sind Texte, die in einem Applet oder Fenster platziert werden können:


    
 
add (new Label ("Beispiel:"))

 
 
Beispiel

Mit den Methoden getText() und setText() kann der Text in einem Label ausgelesen bzw. verändert werden. Labels sind aber statisch, d.h. der Benutzer kann die Texte in einem Label nicht ändern. Einzeilige Textfelder, deren Inhalt der Benutzer ändern kann, nennt man TextField:

    
 
add (new TextField ())

 
 
Beispiel

Mit den Methoden getText() und setText() kann der Text ausgelesen und verändert werden. Mit setColumns() kann die Anzahl der in dem Textfeld dargestellten Zeichen festgelegt werden. Eine weitere wichtige Methode ist setEchoChar(). Diese Methode legt ein Ersatzzeichen fest, welches an Stelle der eingegebenen Zeichen angezeigt wird. Damit lassen sich Passworteingaben realisieren.

    

Ein Beispiel für Labels und Textfelder finden Sie in Kapitel 3.8 unter Applet anzeigen (Karte 1).
