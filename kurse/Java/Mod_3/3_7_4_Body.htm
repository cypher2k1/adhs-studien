TextArea

Eine TextArea ist wie das einzeilige TextField ebenfalls ein Textfeld, allerdings kann der Benutzer hier mit einem Carriage Return eine neue Zeile beginnen.

  
Definition

Die folgende Zeile erzeugt ein Eingabefeld mit 4 Zeilen und 20 Spalten:

  
 
add (new TextArea (4, 20))

 

 
Beispiel

Ein Beispiel für eine TextArea finden Sie in Kapitel 3.8 unter Applet anzeigen (Karte 4).

    

Buttons

    

Mit der folgenden Zeile wird ein Button mit der Beschriftung "button 1" erzeugt:

    
 
add (new Button ("button 1"))

 
 
Beispiel

Ein Beispiel für Buttons finden Sie in Kapitel 3.8 unter Applet anzeigen.
