Checkbox und CheckboxGroup

Mit der folgenden Anweisung wird eine neue Checkbox erzeugt, die der Benutzer mit einem Häkchen selektieren kann. Zunächst ist diese neue Checkbox nicht selektiert.

    
 
add (new Checkbox ( "Rot", false))

 
 
Beispiel

Mit den Methoden getState() und setState() kann auch später der Zustand der Checkbox (selektiert oder nicht selektiert) ausgelesen oder verändert werden.

    

Zu derselben Klasse gehören in Java auch die Optionsfelder. Hier kann der Benutzer nach dem Entweder-Oder-Prinzip genau eine Option auswählen.
So wird eine CheckboxGroup erzeugt:

  
Definition
 
CheckboxGroup group = new CheckboxGroup();

add (new Checkbox ( "Gut", group, true));
add (new Checkbox ( "Nicht Gut", group, false));

 
 
Beispiel

Ein Beispiel für Checkboxen und Optionsfelder finden Sie in Kapitel 3.8 unter Applet anzeigen (Karte 1 und Karte 2).
