Choice

Ein Choice ist ein Popup-Menü, aus dem der Benutzer jeweils einen Eintrag auswählen kann. Nach dem Erzeugen des Menüs werden die einzelnen Einträge hinzugefügt:


  
Definition
 
Choice ziele = new Choice();

ziele.addItem ("Paris");
ziele.addItem ("New York");
ziele.addItem ("London");
add (ziele);

  
 
Beispiel

Den Eintrag, den der Besucher ausgewählt hat, können Sie mit der getSelectedItem()-Methode abfragen. Vorgaben können Sie mit select() machen.

    

Ein Beispiel für ein Popup-Menü finden Sie in Kapitel 3.8 unter Applet anzeigen (Karte 3).
