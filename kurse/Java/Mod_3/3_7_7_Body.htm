List

Eine Liste (engl.: list) besteht aus einer Reihe von Einträgen, von denen der Benutzer einen oder mehrere selektieren kann. Mit den folgenden Anweisungen wird zunächst eine Liste mit vier Elementen erzeugt. Von den vier angezeigten Elementen können mehrere Elemente gleichzeitig ausgewählt werden.


 
Definition
 
List l = new List(4, true);

l.add ("Spinat");
l.add ("Schinken");
l.add ("Brokoli");
l.add ("extra Käse");
add (l);

  
 
Beispiel

Mit der Methode getSelectedItems() können Sie die Liste mit den vom Benutzer ausgewählten Einträgen auslesen.

    

Ein Beispiel dafür finden Sie in Kapitel 3.8 unter Applet anzeigen (Karte 5).
