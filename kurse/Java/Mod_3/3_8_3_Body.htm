BorderLayout

Das BorderLayout ist ein LayoutManager, der die zur Verfügung stehende Fläche in fünf Bereiche unterteilt: Norden, Süden, Osten, Westen und die Mitte. In jedem dieser Bereiche können Elemente, wie zum Beispiel Buttons, platziert werden:

  
Definition
 
    
 
setLayout (new BorderLayout ());

add (new Button ("button 1"), "North");
add (new Button ("button 2"), "West");
add (new Button ("button 3"), "South");
add (new Button ("button 4"), "East");
add (new Button ("button 5"), "Center");
