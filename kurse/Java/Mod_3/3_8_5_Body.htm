GridBagLayout (2)

Als nächstes wird ein Button erzeugt, der bei dem Rasterpunkt 3,0 beginnt, sich über genau eine Rasterzelle erstreckt und dessen Höhe sich an die Rasterhöhe anpasst. Die Breite ist unabhängig vom Raster.

    
 
c.gridx = 3;
c.gridy = 0;
c.weightx = 0;
c.weighty = 1.0;
c.gridwidth = 1;
c.gridheight = 1;

b = new Button ("button 2");
l.setConstraints (b, c);
add (b);
