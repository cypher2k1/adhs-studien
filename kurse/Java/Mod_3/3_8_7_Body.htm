CardLayout (2)

Da immer nur eine Karte angezeigt wird, müssen Sie außerdem noch eine Navigationsleiste oder Buttons programmieren.

    

Mit den Methoden first() und last() springen Sie an die erste bzw. letzte Karte, die Methoden next() und previous() springen jeweils eine Karte vor oder zurück. Mit der Methode show() können Sie eine bestimmte Karte anspringen.
