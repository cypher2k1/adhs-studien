Das Anwendungsprogramm (2)

Die Anweisung, die ein Programm beendet, kennen Sie bereits:


    
 
if (cmd.equals("Beenden"))
   System.exit (0);

  
 
swingapp2.java

Das "Beenden"-Kommando beendet das Programm und das "Info..."-Kommando erzeugt eine kleine Dialogbox.


    
 
if (cmd.equals("Info..."))
   JOptionPane.showMessageDialog (
      null,
      "Guten Tag!" +
      " - Ich bin Ihr erstes Swing Programm/"
   );
