Tastaturkürzel

Als Mnemonic werden in Java die Tastaturkürzel bezeichnet, mit denen Sie Menüs und Buttons ausstatten können:

  
Definition
 
  
 
mFile.setMnemonic ('D');
cmdInfo.setMnemonic ('N');
cmdQuit.setMnemonic ('B');

 
 
SwingGUI3.java

Bei den Kürzeln handelt es sich um einzelne Buchstaben, die im Menüeintrag vorkommen müssen. Sie werden beim Zeichnen des Menüs unterstrichen.

    
  So sieht es aus.
   

Anstatt das Menü (oder den Eintrag) mit der Maustaste anzuklicken, kann der Benutzer so Ihr Programm mit der Alt-Taste auch über die Tastatur bedienen.
