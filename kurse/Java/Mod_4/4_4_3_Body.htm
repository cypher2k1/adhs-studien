Tooltipps anzeigen

Tooltipps sind kleine Hilfetexte, die dem Benutzer angezeigt werden, wenn er mit dem Mauszeiger in einem Menüeintrag oder Dialog kurz verweilt.

  
Definition
 
cmdInfo.setToolTipText ("Zeigt Informationen an...");
cmdQuit.setToolTipText ("Beendet das Programm");

 

 
SwingGUI3.java

Mit diesen Tipps können Sie dem Benutzer zusätzliche Informationen darüber geben, was hinter dem jeweiligen Mausklick "steckt".

    
  So sieht es aus.
