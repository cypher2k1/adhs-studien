Scrollpanes

In den vorherigen (AWT-) Beispielen wurden alle Elemente stets innerhalb des Fensters angezeigt. Das ist weder zwingend notwendig, noch immer möglich. Wenn beispielsweise Bilder angezeigt werden sollen, die größer sind als das Fenster auf dem Bildschirm.

    

Als Container für diese Elemente dient hier eine Scrollpane. Sie umfasst eine beliebig große Fläche, von der immer ein Ausschnitt in einem Fenster angezeigt wird.

    

Die Scrollpane ist sowohl Container für Elemente, die dargestellt werden sollen, als auch Element, welches in einen Container eingetragen wird (s. vorherige Seite).


    
 
JScrollPane scrollPane;
scrollPane = new JScrollPane ();
topPanel.add (scrollPane, BorderLayout.CENTER);
