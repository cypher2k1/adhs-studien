Inhalte von Tabellen

Gegenüber den AWT-Klassen sind Tabellen als Komponenten eine weitere wichtige Neuerung der Swing-API.

    

Swing-Tabellen bestehen aus Zeilen und Spalten bzw. einer Matrix aus Zellen, in denen Werte stehen. Die Spalten besitzen (optional) einen String als Namen oder Titel: Name, Vorname und Telefon:

    
 
String columnNames [] = { "Name", "Vorname", "Telefon"};

Die Inhalte der Tabelle sind im einfachsten Fall ebenfalls Strings. Sie können beispielsweise durch ein zweidimensionales Array festgelegt werden:

    
 
String dataValues[][] = {
   { "Meier", "Werner", "069 - xxx" },
   { "Müller", "Frank", "089 - xxx" },
   { "Müller", "Sybille", "089 - xxx" },
   { "xyz", "Sabine", "02331 - xxx" }
};
