RSS feeds in Java
In diesem Kapitel soll nun ein Programm geschrieben werden, welches die aktuellen Schlagzeilen von der Spiegel-Internetseite lädt und auf dem Bildschirm ausgibt. In solchen Java-Programmen können Sie die RSS feeds in zwei Schritten einbauen:    
  Herunterladen des Nachrichtentexts
  Auslesen der gewünschten Informationen, zum Beispiel die Überschriften bzw. Schlagzeilen.

   

Der erste Schritt, das Herunterladen der aktuellen Schlagzeilen, ist einfach: Wir brauchen dazu:

    
  ein Objekt der Klasse URL aus dem Package java.net
  einen InputStream
  einen InputStreamReader
  einen BufferedReader
