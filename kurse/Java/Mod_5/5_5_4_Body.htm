Einladen eines RSS feeds

In unserem Beispiel werden die Inhalte der Internetadresse http://www.spiegel.de/schlagzeilen/rss/index.xml Zeile für Zeile ausgelesen (vgl. Kapitel 6.2) und zu einem großen String zusammengesetzt:

   
 
URL       url;
String    adresse = "http://www.spiegel.de/schlagzeilen
                                          /rss/index.xml";

try {
   url  = new URL (adresse);
   InputStream        is  = url.openStream();
   InputStreamReader  isr = new InputStreamReader( is );
   BufferedReader      br = new BufferedReader( isr );

   String zeile;
   while(( zeile = br.readLine() ) != null ){
      content = content + zeile;
   }

} catch (Exception e) {  }
      System.out.println (e);
}
System.out.println (content);

  


 
Schlagzeilen.java

In der letzten Zeile wird dann dieser String auf dem Bildschirm ausgegeben (siehe nächste Seite).

Es handelt es sich um eine XML-Datei (vgl. Kapitel 6.8). Alles was wir an dieser Stelle des Kurses hierzu wissen müssen ist, dass die Schlagzeilen in den so genannten XML-Tags <title> .. </title> stehen.
