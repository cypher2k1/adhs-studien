Die Verfahren

Unter Serialisierung versteht man das Konvertieren eines Objekts in eine lineare Folge von Bytes. Die Byte-Folge kann anschließend in eine Datei geschrieben werden oder über ein Netzwerk kopiert werden.

 
Definitionen

Das JDBC (Java Database Connectivity) beinhaltet die Definitionen von Funktionen, mit denen auf relationale (SQL-) Datenbanken zugegriffen werden kann. Verschiedene Datenbankhersteller bieten JDBC-Schnittstellen für Ihre Datenbanken an. So können JDBC-Programme auf Datenbanken zugreifen, ohne zu wissen, in welcher Datenbank die Daten gespeichert werden.
