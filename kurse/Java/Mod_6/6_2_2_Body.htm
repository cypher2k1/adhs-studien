Einleitung

In Java laufen die Ein- und Ausgabe von Daten häufig über eine Hierarchie mehrerer Objekte, die in Schichten oder "Ebenen" angeordnet sind. Auf einer unteren Ebene gibt es Objekte, die einzelne Bytes verarbeiten (Einladen oder Abspeichern). Darüber folgt eine Ebene mit Objekten, die ein Objekt der darunter liegenden Schicht benutzen, um ganze Texte zeilenweise zu verarbeiten.

    
Dieses Verfahren ist anfangs etwas umständlich, bietet jedoch den Vorteil, dass man die Objekte auf einer unteren Ebene austauschen kann, ohne die Anwendung mit den darüber liegenden Objekten verändern zu müssen.
