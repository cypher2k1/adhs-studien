Einladen von Texten

Das Einlesen von Texten beruht auf einem ähnlichen Prinzip wie das Abspeichern. Zunächst wird ein Objekt erzeugt, das auf der unteren Ebene Daten aus der Datei liest.

    
 
FileReader f = new FileReader ("remember.txt");

 
 
Lesen.java


Auf der nächst höheren Ebene wird damit ein Objekt erzeugt, das Texte auslesen kann.

    
 
BufferedReader in = new BufferedReader (f);

  
   


Mit diesem Objekt wird schließlich der Inhalt der Datei zeilenweise ausgelesen.

    
 
String s;

while ((s = in.readLine ()) != null) {
   System.out.println (s);
}
in.close ();
