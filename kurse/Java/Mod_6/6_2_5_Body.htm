Streams

Als Streams bezeichnet man in Java ein weiteres Konzept für geschachtelte Ein- und Ausgabeobjekte. Sie heißen nun nicht mehr Reader und Writer, sondern InputStreams und OutputStreams und sind nicht mehr auf Strings beschränkt.

  
Definition

So gibt es zum Beispiel auf der unteren Ebene die Klasse FileOutputStream zum Schreiben von Daten in eine Datei. Darüber gibt es die Klasse ZipOutputStream zum Komprimieren von Daten (und Schreiben der Daten in den enthaltenen Stream).

   
 
FileOutputStream f   = new FileOutputStream ("remember.zip");
ZipOutputStream  zip = new ZipOutputStream (f);

  
 
SpeichernZIP.java


Dieser Stream kann nun verwendet werden, um einen Text unter einem bestimmten Namen (test.txt) in einem ZIP-Archiv abzulegen.

    
 
String s = "Dieses ist ein Test!";

zip.putNextEntry (new ZipEntry ("test.txt"));
zip.write (s.getBytes (), 0, s.length ());
zip.close ();
