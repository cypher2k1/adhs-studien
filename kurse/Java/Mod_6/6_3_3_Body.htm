Das Datenmodell: Personen

Im Mittelpunkt der Universität stehen Personen. Sie werden der Einfachheit halber nur durch ihren Namen und Vornamen beschrieben.

    

Dazu wird die Klasse für die Personen, wie die folgenden Klassen auch, in einem neuen Paket abgelegt:

    
 
package UniBeispiel;
[...]
public class Person implements Serializable {
   public String  m_name;
   public String  m_vorname;
[...]
}

 

 
Person.java
Bitte beachten Sie, dass die Attribute m_name und m_vorname hier als public deklariert werden. Eine echte Anwendung sollte diese Informationen kapseln und nach außen nur Methoden zum Lesen und Bearbeiten der Eigenschaften anbieten. Vergleichen Sie dies mit den Eigenschaften von JavaBeans in Kapitel 7.1.
 
Hinweis


Die Klasse implementiert das Interface java.io.Serializable. Die Erläuterungen hierzu folgen nach der Vorstellung des Datenmodells.
