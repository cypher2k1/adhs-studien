Einleitung

Das JDBC (Java Database Connectivity) ist eine Schnittstelle, die es Java-Programmen ermöglicht, mit standardisierten Methoden auf relationale Datenbanken zuzugreifen. Im Idealfall merken die Programme gar nicht, in welchen Datenbanken Ihre Daten abgelegt werden.

  
Definition
Viele Hersteller von Standard-Datenbanken bieten Implementierungen des JDBC an. Dabei handelt es sich um so genannte Call Level Interfaces, d.h. Java-Programme rufen die Datenbank auf und übersenden ihr die SQL-Anweisungen in Form von Strings. Mit weiteren Aufrufen können schließlich die Ergebnisse abgefragt werden.

 
Hinweis

Ein Beispiel für eine JDBC-Implementierung stellt die JDBC-ODBC-Bridge da. Hier werden die JDBC-Aufrufe auf die ODBC-Schnittstelle umgeleitet, die bei Microsoft Windows PCs sehr verbreitet ist und auch für MacOS und LINUX/UNIX erhältlich ist. Damit lassen sich alle Datenbanken, zu denen es eine ODBC-Schnittstelle gibt, aus Java heraus aufrufen. Und das sind fast alle.
