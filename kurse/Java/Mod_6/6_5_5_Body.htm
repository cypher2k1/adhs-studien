SQL - Abfragen und Einfügen

Mit dem INSERT-INTO-Kommando können in SQL Werte in eine Tabelle eingefügt werden. So kann zum Beispiel eine neue Uni erzeugt werden:

    
 
INSERT INTO Uni (ID, f_name) VALUES (1, "Uni. XY");

  
 
Beispiele


Dabei werden hinter dem INSERT INTO Uni zunächst die Spalten angegeben, in die die tatsächlichen Werte eingetragen werden sollen. Da der Wert der ID-Spalte von der Datenbank selber vergeben wird, braucht dieser nicht mehr angegeben zu werden:

   
 
INSERT INTO Uni (f_name) VALUES ("Uni. XY");       

  

   

Mit der SELECT-Anweisung können die Werte aufgerufen werden, die in einer Tabelle stehen. Die folgende Anweisung liefert alle Werte in der Tabelle zurück.

    
 
SELECT * FROM Uni;                           

 
   


Folgende Einschränkungen sind möglich: Statt des Sterns (*) können genau die Spalten angeben werden, die angezeigt werden sollen. Die Einfügung einer WHERE-Klausel erlaubt die selektive Abfrage bestimmter Datensätze. Mit der folgenden Anweisung wird die ID der zuvor eingefügten Universität abgefragt:

    
 
SELECT ID FROM Uni WHERE f_name = "Uni. XY";
