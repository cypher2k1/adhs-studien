ODBC und Microsoft Access

Ist die Datenbank in Access angelegt, muss sie als ODBC-Datenquelle im System bekannt gemacht werden. Hierzu existiert unter Start / Einstellungen / Systemsteuerung / Verwaltung bzw. in älteren Windows-Versionen unter Start / Einstellungen / Systemsteuerung der Eintrag Datenquellen (ODBC)

   

Mit diesen Einstellungen wird die Datenbank unter dem Namen UNI_DB_ODBC im System bekannt gemacht (Access-Datenbanken werden in Dateien mit der Endung .mdb gespeichert. In unserem Beispiel ist der Speicherort d:\work\Kap6.

    

Damit kann in dem nächsten Schritt eine Applikation erstellt werden, die eine Verbindung zu der Datenbank aufbaut und dort Daten ablegt bzw. abfragt.
