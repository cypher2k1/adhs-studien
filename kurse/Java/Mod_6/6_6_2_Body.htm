Die Verbindung aufbauen

Nach dem Anlegen der Datenbank wird die JDBC-Anwendung erstellt. Der erste Schritt ist der Aufbau einer Verbindung zur Datenbank:

    
 
Connection c = null;

try {
  Class.forName ("sun.jdbc.odbc.JdbcOdbcDriver");
  c = DriverManager.getConnection ("jdbc:odbc:UNI_DB_ODBC");
[...]

 

 
UniDB.java
Zunächst wird die Klasse für die JDBC-OBDC-Bridge über die Class.forName ("...")-Anweisung geladen; danach wird die spezifische ODBC-Datenquelle ausgewählt (hier: UNI_DB_ODBC). Die Anweisungen zum Verbindungsaufbau unterscheiden sich in den Parametern, die den Methoden forName() und getConnection() übergeben werden. Sie sind für jeden JDBC-Treiber verschieden. Die nötigen Werte entnehmen Sie im Zweifelsfall der Dokumentation Ihrer Datenbank.
