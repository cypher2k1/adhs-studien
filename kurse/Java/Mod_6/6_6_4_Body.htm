Abfragen aufbauen, abschicken und auswerten

Nun wird eine SELECT-Anweisung definiert, an die Datenbank geschickt und dort ausgeführt:

    
 
ResultSet rs;

sql = "SELECT ID FROM Uni WHERE f_name = '";
sql = sql + u.m_name;
sql = sql + "'";

rs = s.executeQuery (sql);

 
 
UniDB.java


Rückgabe dieser Query ist ein Objekt vom Typ ResultSet, also die Ergebnismenge der Datenbank. Ein ResultSet beinhaltet alle zurück gelieferten Datensätze. Jeweils einer davon wird mittels eines Cursors als "aktueller Datensatz" markiert, dessen Werte mit entsprechenden Methoden ausgelesen werden können. Zunächst steht der Cursor noch vor dem ersten Datensatz. Mit der Anweisung next () wird er um eine Position nach vorne geschoben. Die Funktion liefert true zurück, solange an der aktuellen Position noch ein gültiger Datensatz existiert. Nach dem letzten Datensatz wird false zurückgeliefert.

    

Die folgende Anweisung prüft, ob überhaupt ein gültiger Datensatz vorhanden ist:

    
 
if (rs.next ())
[...]

 
 
UniDB.java
 
    
Alternativ können mit while (rs.next ()) alle gefundenen Datensätze durchlaufen werden.
