XML-Parser

Zum Einlesen eines XML-Dokuments braucht man einen XML-Parser. Dies ist ein Programm, eine Bibliothek oder Klasse, die eine Textdatei (im XML-Format) einliest und den Inhalt auf Übereinstimmung mit der angegebenen DTD testet. Als Ergebnis wird die Datei in einer strukturierten Form zu Verfügung gestellt, die von einem Java-Programm benutzt werden kann.

  
Definition
Zum Speichern des eingelesen Dokuments in einer Datenstruktur, die für Anwendungsprogramme nutzbar ist, verwenden viele Parser das DOM (Document Object Model) des W3C-Konsortiums. Die Internetadresse lautet:

http://www.w3.org/DOM/

Das DOM beschreibt ein Interface, mit dem auf die Strukturen eines solchen Dokuments zugegriffen werden kann.
