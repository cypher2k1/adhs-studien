Implementierung eines Integer-Textfeldes (2)

Der nächste Schritt ist ebenfalls einfach: In Abhängigkeit dieses Werts wird entweder nichts getan und der Event wird normal ausgeführt, wenn es sich um eine Ziffer oder um das Löschen eines Zeichens handelt. Oder der Event wird hier auf der Stelle "konsumiert", sodass er in anderen Methoden nicht mehr verarbeitet werden kann. Der Tastendruck wird sozusagen "runtergeschluckt" und nicht weiter ausgewertet.

    
 
public void processKeyEvent (KeyEvent event) {

  char c = event.getKeyChar ();

  if ((c >= '0' && c <= '9') ||
      (c == KeyEvent.VK_DELETE) ||
      (c == KeyEvent.VK_BACK_SPACE)) {

      // ... tue nichts, d.h.
      // der Event wird normal verarbeitet

  } else {

      // lösche den Event, ohne Verarbeitung
      event.consume ();
  }
}
