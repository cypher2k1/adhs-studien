Die Eigenschaften "unserer" JavaBean

Die JavaBean, die wir hier programmieren, wird von der Klasse TextField abgeleitet, die wir in Kapitel 3.7 bereits kennen gelernt haben. Außerdem haben wir Ihnen in Kapitel 7.1 gezeigt, dass eine JavaBean bestimmte Eigenschaften hat, wenn es entsprechende get()- und set ()- Methoden gibt (auch getter- und setter- Methoden genannt). Daraus folgt, dass diese JavaBean schon jetzt eine Eigenschaft Text hat, denn mit getText () und setText () können Sie den Text des Feldes auslesen bzw. verändern.

    

Allerdings sollte die JavaBean für ein Integerzahlen-Textfeld etwas komfortabler sein, etwa durch eine neue Eigenschaft IntegerValue. Nach der nun bekannten Theorie muss die JavaBean also eine Methode namens getIntegerValue besitzen, welche die aktuelle Zahl in dem Textfeld zurückliefert, und eine Methode setIntegerValue, die eine Zahl als Text in das Feld einträgt.

   

Die erste Methode lautet:

    
 
public int getIntegerValue () {

   int result = 0;

   try {

      result = Integer.parseInt (this.getText ());
   }
   catch (Exception e) {}

   return result;
}
