Das Erzeugen von JavaBean-Instanzen

In das Layout kann dann eine Instanz der JavaBean eingetragen werden, wie jede andere Komponente auch:

    
 
[...]
setLayout (new FlowLayout (FlowLayout.LEFT, 10, 10));

IntegerTextField1 tf1 = new IntegerTextField1 ();

 
 
Anwendung1.java


Da sie in diesem Fall alle Eigenschaften und Methoden von TextField erbt, können die Methoden benutzt werden, um die Anzahl der Spalten festzulegen, bevor die JavaBean in das Layout eingetragen und angezeigt wird. So wird ein leeres Textfeld erzeugt:

    
 
tf1.setColumns (10);                                
add (tf1);
[...]

  
 
Anwendung1.java


Zusätzlich können die neu definierten JavaBean-Methoden benutzt werden, um zum Beispiel einen Startwert für das neue Feld festzulegen. Mit der zusätzlichen Anweisung setIntegerValue (0); würde der Wert 0 als Startwert in das Feld eingetragen werden.
