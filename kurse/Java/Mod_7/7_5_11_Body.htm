Die JSP-Seite im Detail (2/4)

Das dritte Beispiel für Java-Code in dieser JSP-Seite ist die Ausgabe von Text mit der println()-Methode:

    
 
<%=
    System.out.println( "Ausgabe auf dem Server" );
    out.println( "Ausgabe in die HTML Seite" );
%>

  
 
beispiel.jsp


System.out.println([...]) gibt einen Text auf dem Webserver aus. Der Webserver Tomcat zeigt diese Zeile jedoch nicht direkt auf dem Bildschirm an, sondern schreibt sie in eine log-Datei. Diese Datei wird automatisch in einem Unterverzeichnis namens logs angelegt und heißt stdout, gefolgt von dem aktuellen Datum. Den Inhalt der Datei können Sie sich unter MS DOS mit dem Befehl more anschauen:

    
  So sieht es aus.
   


Vergleichen Sie dies doch einmal mit der Ausgabe im Browser (zweite Zeile von oben):

    
  So sieht es aus.
   


Dazu wird das Objekt out verwendet, welches implizit in jeder JSP-Seite definiert wird. Sie können dieses Objekt einsetzen, um Text in die generierte HTML-Seite zu schreiben.
