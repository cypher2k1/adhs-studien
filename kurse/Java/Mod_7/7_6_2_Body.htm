Java Enterprise Edition

Die Java Enterprise Edition (Java EE) ist die Architektur zum Erstellen und Ausführen von Java-Anwendungen auf einem Webserver. Dies beinhaltet zum Teil in diesem Kurs bereits vorgestellte Konzepte wie die Sprache Java als solches, JavaBeans, Datenbankanbindung über JDBC, User Interfaces über JSPs etc.

  
Definition
Aber die Java EE-Spezifikation geht weit darüber hinaus. Sie beinhaltet Konzepte für Transaktionsmanagement, Sicherheit und Installationssupport (engl.: Deployment). Neben dem JDK braucht man zum Entwickeln und Betreiben von Java EE-Anwendungen einen so genannten Anwendungsserver wie beispielsweise JBoss (ein Open-Source-Projekt) oder IBMs WebSphere.
