Aktivitätsdiagramme

Mit Aktivitätsdiagrammen lassen sich die einzelnen Schritte verdeutlichen, aus denen ein Use Case besteht. Sie werden als einzelne Aktivitäten bezeichnet, beginnend mit einem Start und mit einem Ende. Dazwischen kann es Verzweigungen geben und mehrere parallel ablaufende Aktivitäten.

  
Definition

Die Aktivitäten werden in den Diagrammen durch Rechtecke mit abgerundeten Ecken symbolisiert. Von jeder Aktivität geht jeweils ein Pfeil zur nächsten Aktivität. Ausgefüllte Kreise bedeuten Anfang und Ende dieser Kette, wobei um das Ende ein weiterer Kreis gezeichnet wird.
