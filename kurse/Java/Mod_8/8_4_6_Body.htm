Klassen und Attribute

Im folgenden Beispiel wurden in Argo/UML der Klasse Person zwei Attribute hinzugefügt: nachname und vorname. Die Syntax entspricht in diesem Beispiel der Syntax von Argo/UML.

    

Abbildung: Klassendiagramm - Klassen und Attribute
  
Beispiel


Aus diesem Diagramm wird der zugehörige Java-Code generiert:

    
 
public abstract class Person {

   protected String vorname;
   protected String nachname;
}

  

 
Beispiel
Laut UML-Konvention sollen die Namen der Attribute mit einem Kleinbuchstaben beginnen. Falls es sich um deutsche Namen handelt, wird jedoch wegen der besseren Lesbarkeit oft ein Großbuchstabe verwendet.
