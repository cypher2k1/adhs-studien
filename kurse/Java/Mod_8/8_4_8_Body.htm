Operationen mit Argumenten
Sollen die Argumente bei der Operation mit angegeben werden, stehen diese in Klammern. Mehrere Argumente werden durch Kommata getrennt. Allerdings weichen auch hier viele Tools von der offiziellen Notation ab.
 
Hinweise
Zu jedem dieser Argumente kann zunächst festgelegt werden, ob es sich um Eingabe- oder Ausgabeparameter handelt. Hierzu beginnt die Angabe eines Arguments mit dem Wort in, out oder inout. Dem folgt der Name des Arguments, der Datentyp und ein Defaultwert, für den Fall, dass kein Wert als Parameter angegeben wird. Nicht alle Programmiersprachen unterstützen diese Konzepte. So gibt es in Java keine transienten Argumente, sondern nur Eingabeparameter. Auch werden von Java keine Defaultwerte unterstützt. Aus diesem Grund werden diese Features von UML nicht von allen Tools unterstützt.
