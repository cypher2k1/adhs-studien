javax.swing
    Enthält eine Reihe von lightweight (in Java programmierten) -Komponenten für grafische Benutzeroberflächen, die auf allen Betriebssystemen möglichst gleich aussehen und arbeiten.
   
  Zum Nachlesen:
Swing		    
  javax.swing.border
    Enthält Klassen und Interfaces für das Zeichnen spezieller Ränder um Swing-Komponenten.
   
   
  javax.swing.colorchooser
    Enthält Klassen und Interfaces, die vom JColorChooser zur Farbauswahl benutzt werden.
   
   
  javax.swing.event
    Enthält die Events, die von Swing-Komponenten ausgelöst werden.
   
   
  javax.swing.filechooser
    Enthält Klassen und Interfaces, die von dem JFileChooser zum Daten auswählen benutzt werden können.
   
   
  javax.swing.plaf
    Enthält ein Interface und abstrakte Klassen, die von Swing für das Pluggable Look & Feel benutzt werden.
